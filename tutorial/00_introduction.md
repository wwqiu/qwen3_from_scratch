# 第 0 章：导言 —— 拆解LLM推理黑盒

这是一份通过不依赖任何第三方代码一步一步实现qwen3推理引擎，从而拆解LLM推理黑盒的开发教程。

如果你：
- 渴望揭开 LLM 底层运行的神秘面纱
- 对炙手可热的 AI Infra 领域感兴趣却苦于找不到系统的入门门径
- C++初学者想找一个硬核且充满成就感的极客项目

那这份教程就是为你量身定制的。

### 你需要准备什么？

只要你具备以下基础，这个教程对你来说将**毫无门槛**：
- ✅ **C++ 基础**：熟悉指针、面向对象（类）、以及基本的 STL 容器。
- ✅ **一点点线性代数**：知道最基本的矩阵乘法和向量运算是怎么回事。
- ✅ **神经网络常识**：对大模型有个概念，听过“权重”、“激活函数”这些基本名词。
- ⚠️ **完全不需要**：你不需要有任何 PyTorch/TensorFlow 等深度学习框架的使用经验，也完全不需要懂复杂的 CUDA 编程。

### 你将收获什么？
1. **理解 LLM 推理流程**
   - 从文本输入到 token 输出的完整过程
   - 每个模块的作用和数学原理

2. **掌握核心算子实现**
   - Embedding、RMSNorm、Attention、MLP 等
   - 理解 RoPE、GQA、SwiGLU 等技术

3. **能够独立实现简单的推理引擎**
   - 加载模型权重
   - 实现前向传播
   - 进行文本生成

4. **为后续优化打下基础**
   - 理解性能瓶颈在哪里
   - 知道如何进行优化（量化、并行等）
  

在这份教程中，我们将遵循以下原则：
1. **不炫技**：拒绝晦涩的 C++ 高级模板，代码怎么直观怎么写，把认知负担留给算法本身。
2. **轻量级**：没有庞大的第三方依赖，一台普通的轻薄本就能跑起 Qwen3-0.6B。
3. **图文结合**：把复杂的矩阵运算和维度变换，变成看得见、摸得着的图表和代码。


### 路线图


*   **第 1 章：架构总览** —— 深入拆解 Transformer Decoder-only 宏观结构与推理流水线。
*   **第 2 章：环境集成** —— 配置 C++ 开发工具链，基础库依赖。
*   **第 3 章：张量 (Tensor)** —— 实现支持多维索引的张量容器，管理权重与激活值的内存分配。
*   **第 4 章：分词引擎 (Tokenizer)** —— 深入 BPE 编码原理，实现字节级文本与 Token ID 的双向映射逻辑。
*   **第 5-7 章：核心算子 (Operators)** —— 逐行实现 RMSNorm、RoPE 位置编码及 Multi-Head Attention 等核心数学模块。
*   **第 8 章：模型集成 (Qwen3Model)** —— 封装完整前向传播链路，实现多层 Decoder Block 的串联与权重加载。
*   **第 9 章：推理循环 (Main Loop)** —— 编写自回归生成逻辑，跑通完整的端到端对话流程。
*   **第 10 章：性能优化 (KV Cache)** —— 引入键值缓存机制，消除历史序列重复计算，提升长文本推理效率。

让我们开始吧！
